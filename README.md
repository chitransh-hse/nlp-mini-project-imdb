# NLP Mini Project — IMDb Sentiment Analysis

## Overview
This project implements a classical NLP pipeline for sentiment classification using the IMDb dataset.

The pipeline includes:
- Text preprocessing
- TF-IDF feature extraction
- Logistic Regression and Neural Network models
- Performance comparison

## Dataset
IMDb movie reviews dataset (binary sentiment classification).

## Methods
- TF-IDF (unigram and bigram)
- Logistic Regression
- Feedforward Neural Network

## Results
The models are evaluated using accuracy and classification metrics. A comparison between feature representations and models is provided.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run notebook:
   jupyter notebook notebook.ipynb
